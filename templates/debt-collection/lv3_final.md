# 督促テンプレート Lv3：最終確認

---

## 重要事項（省略・削除禁止）

```
★ オーナー承認必須
  このテンプレートを使用する前に、必ず徳田（オーナー）の承認を得てください。
  承認なしの送信は禁止です。

★ 法的・心理的圧迫表現禁止
  「法的措置」「弁護士」「訴訟」「差し押さえ」等の脅迫的・圧迫的な
  表現はこのテンプレートに含めてはいけません。
  表現を修正する場合も、圧迫表現を追加することは禁止です。

★ 送信は人間のみ
  AIは文章生成のみを行います。
  このメッセージの送信は必ず人間（担当者またはオーナー）が行ってください。
  AIによる自動送信は絶対禁止です。
```

---

**使用条件**: 支払期日から14日超経過、または優先度Sかつ早急対応が必要  
**送信手段**: LINE（推奨）または電話（担当者判断）  
**送信者**: 人間のみ（オーナー承認後）  
**前提**: Lv1・Lv2の記録が必ずあること。なければ上位レベルから対応

---

## 送信テキスト

```
{{customer_name}} 様

お世話になっております。{{business_name}}の{{owner}}です。

先日よりご連絡させていただいておりました
¥{{amount}} のお支払いについて、
現時点でご入金の確認が取れておりません。

大変恐れ入りますが、下記を最終期限とさせていただきます。

■ 最終期限：{{final_deadline}}（★人間が日付を設定★）
■ ご請求金額：¥{{amount}}
■ お振込先：[★人間が口座情報を追記★]

期限までのご対応が難しい場合は、お早めにご相談ください。
分割等、可能な範囲でご相談に応じます。

お手数をおかけしますが、どうぞよろしくお願いいたします。
```

## 差し込み変数

| 変数名 | 内容 | 必須 | 備考 |
|--|--|--|--|
| `{{customer_name}}` | 顧客名 | 必須 | |
| `{{business_name}}` | 事業名 | 必須 | |
| `{{owner}}` | 担当者名 | 必須 | |
| `{{amount}}` | 未収金額（円） | 必須 | カンマ区切り |
| `{{final_deadline}}` | 最終期限 | 必須 | **人間が設定**（最低5営業日後推奨）|

---

## 人間確認チェックリスト（送信前に全項目確認・省略不可）

- [ ] **オーナー（徳田）の承認を口頭または書面で得たか**
- [ ] Lv1・Lv2の送信記録が DEBT_COLLECTION_LOG にあるか
- [ ] `{{final_deadline}}` の日付は現実的か（最低5営業日後）
- [ ] 金額が正しいか
- [ ] 口座情報を追記したか
- [ ] 圧迫的な表現が含まれていないか（再確認）
- [ ] 顧客が支払い困難な事情を申告している場合、分割交渉を先に提示したか
- [ ] 送信先が正しいか

---

## Lv3送信後のルール

- `{{final_deadline}}` を過ぎた後の対応は**必ずオーナーが判断**する
- AIはLv3送信済み案件に対してそれ以上の督促文を自動生成しない
- 次のアクション（交渉継続・断念・専門家相談等）はオーナーが決定する

---

## 送信後の記録（人間が実施）

DEBT_COLLECTION_LOG の以下を更新：
- `N列（Lv3送信日）`：送信日を入力
- `P列（最終期限）`：`{{final_deadline}}` の値を入力
- `K列（状態）`：「Lv3送信」に変更
- `W列（備考）`：オーナー承認日・承認者を記録
