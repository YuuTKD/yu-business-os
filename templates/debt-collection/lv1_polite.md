# 督促テンプレート Lv1：丁寧な確認

**使用条件**: 支払期日から7日未満経過  
**送信手段**: LINE または メール（担当者が選択）  
**送信者**: 人間（AIは生成のみ・送信しない）

---

## 送信テキスト

```
{{customer_name}} 様

お世話になっております。{{business_name}}の{{owner}}です。

先日はご利用いただきまして、誠にありがとうございました。

お支払い期日（{{due_date}}）について確認のご連絡をいたしました。
ご入金済みでございましたら行き違いをお詫び申し上げます。

まだのご場合は、下記金額にてご対応いただけますと幸いです。

■ ご請求金額：¥{{amount}}
■ お振込先：[★人間が口座情報を追記★]

ご不明な点がございましたらお気軽にご連絡ください。
どうぞよろしくお願いいたします。
```

---

## 差し込み変数

| 変数名 | 内容 | 必須 | 備考 |
|--|--|--|--|
| `{{customer_name}}` | 顧客名 | 必須 | |
| `{{business_name}}` | 事業名 | 必須 | Catering / TACHINOMIYA 等 |
| `{{owner}}` | 担当者名 | 必須 | |
| `{{due_date}}` | 支払期日 | 必須 | YYYY年MM月DD日 形式 |
| `{{amount}}` | 未収金額（円） | 必須 | カンマ区切り（例: 50,000）|

---

## 人間確認チェックリスト（送信前に必ず確認）

- [ ] 顧客名・敬称が正しいか
- [ ] 金額が正しいか（合計額・税込/税抜の区別）
- [ ] 口座情報を追記したか
- [ ] 支払期日が正しいか
- [ ] 送信先（LINE/メール）が正しい相手か
