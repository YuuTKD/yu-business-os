# 新規画像 登録チェックリスト（1枚ごとに実施）

> これを全部埋めるまで、その画像は自動投稿に使えません。
> 埋められない項目があれば「保留」フォルダへ移動し、ゆうさん確認リストに追加。

## メタデータ記入（IMAGE_LIBRARYに追記。既存行の編集・削除は禁止）
- [ ] category（food / drink / interior / event / people / menu）
- [ ] image_theme（1画像1テーマ。迷ったら主役の被写体名）
- [ ] product_name（商品が写っている場合のみ。正式メニュー名で）
- [ ] allowed_post_themes（この画像を使ってよい投稿テーマ。最低1・最大5）
- [ ] blocked_post_themes（誤用されやすい隣接テーマを必ず1つ以上。例：サーターアンダギー画像なら"ラフテー"ではなく、逆にラフテー画像に"サーターアンダギー"）
- [ ] caption_keywords（本文照合用 3〜8個）
- [ ] visual_description（写っているものを1文で。「何が・何個・どんな状態・背景」）

## 安全確認
- [ ] 人物が写っている → 掲載許諾を取ったか（未確認なら使用不可のまま保留）
- [ ] お客様の顔・伝票・ナンバー等の写り込みなし
- [ ] 実物と乖離する過度な加工なし

## 技術確認
- [ ] ファイル容量 1.5MB以下（超えていたら圧縮対象リストへ）
- [ ] 縦向き or 正方形
- [ ] GCS化担当への引き継ぎ（gcs_public_url欄は空欄のまま＝GCS化待ちの印。URLを手入力しない）

## 完了条件
上記すべてチェック → 週次点検で「使える画像」に昇格して完了。
