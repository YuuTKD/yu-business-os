# スタッフ向け撮影依頼文テンプレ

## LINE送信用（短文・そのまま送れる形）
```
【撮影依頼】◯◯（担当者名）さん

▼撮ってほしいもの
・【テーマ：例）泡盛（グラス注ぎシーン）】×【5】枚
・【テーマ：例）せんべろセット全体】×【5】枚

▼撮り方（この3つだけ守ってください）
1. 縦向き・明るい時間帯 or 照明の下で
2. 商品が画面の6〜7割を占めるように寄る
3. 背景にお客様の顔・伝票・私物が写らないように

▼提出
撮ったら Google Drive の【フォルダ名】にそのまま入れて、
このLINEに「入れました」と返信してください。

期限：【日付】まで
1枚30秒でOK、加工不要です📷
```

## 撮影NG例（依頼文に添付する画像ルール）
- ピンボケ / 暗い / 商品が小さい（画面3割以下）
- お客様の顔・車のナンバー・伝票・レジ画面の写り込み
- 横向き（Instagram/Threadsは縦 or 正方形優先）
- 加工アプリでの過度な補正（色味が実物と乖離→誇大表現リスク）

## 依頼後のフロー（登録まで）
1. スタッフがDriveに投入 →「入れました」報告
2. 担当がregister_checklist.mdでメタデータ付与
3. GCS化リストに追加（優先順位はimage-library-managerの出力どおり）
4. 週次点検で「使える画像」入りを確認
