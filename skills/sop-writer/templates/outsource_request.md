# 外注依頼文テンプレ（検収条件・報告様式つき）

## 依頼文の型（クラウドソーシング/直接依頼 共通）
```
【依頼】◯◯の作業をお願いします

■ 依頼内容
【作業を1〜2行で。例：営業DM送信前のリスト作成（対象：那覇市内の◯◯業種、50件）】

■ 具体的な作業手順
1.【手順1】
2.【手順2】
3.【手順3】
※迷った場合は作業を止めて質問してください（推測で進めないでください）

■ 納品物と形式
・【例：Googleスプレッドシート（列：店名/住所/IGアカウント/フォロワー数/最終投稿日）】
・納品先：【共有リンク/フォルダ】

■ 検収条件（これを満たしたら完了）
・【例：50件すべてにIGアカウントURLが入っている】
・【例：最終投稿が90日以内のアカウントのみ】
・検収で差し戻しの場合、修正1回まで依頼料金内でお願いします

■ NG事項
・【例：フォロワーを購入しているような不自然なアカウントを含めない】
・個人情報の目的外利用・第三者提供
・作業内容のSNS等への投稿

■ 納期：【日付】
■ 報酬：【ゆうさん承認後に記入】円（税込/税別）
■ 報告：【毎日/完了時】に「本日◯件完了、残り◯件」の形式で【LINE/チャット】へ

ご質問があれば契約前にお願いします。
```

## 検収チェック（発注側・受け取り時）
- [ ] 納品形式が指定どおり
- [ ] 検収条件を全件満たす（サンプル10%を抜き打ち確認）
- [ ] NG事項への抵触なし
- [ ] 報告様式が守られていた（次回継続の判断材料）

## 金額ルール
- 依頼文の金額欄は必ず【ゆうさん承認後に記入】のまま作成
- 相場調査が必要な場合はopen_questionsで「予算上限」を確認
