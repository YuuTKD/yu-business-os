# 作業別・報告テンプレ集（受け手コピペ用）

## 撮影→Drive投入
```
【撮影完了】◯◯（テーマ名）を5枚、
Driveの【フォルダ名】に入れました。
```

## SNS投稿後のスクショ報告
```
【投稿完了】◯月◯日 ◯時 / 媒体：IG or Threads
スクショ添付（投稿画面＋時刻が写っているもの）
特記：（コメント・不具合があれば）
```

## 口コミ依頼の実施報告（TACHINOMIYA/火鍋）
```
【口コミ依頼】本日◯組に案内、うち◯組がその場で投稿。
断られた理由（あれば）：
```

## 営業リスト作成（外注）
```
【進捗】本日◯件完了 / 累計◯件 / 残り◯件
イレギュラー：（該当アカウントなし等があれば行番号つきで）
```

## Drive整理・写真分類
```
【整理完了】◯◯フォルダ：◯枚を分類済みへ移動。
分類できなかったもの◯枚 →「保留」フォルダに入れました（判断お願いします）
```

## Beauty施術写真の補充（美優）
```
【施術写真】◯◯メニュー ◯枚をDriveへ。
お客様許諾：全件取得済み（同意書：あり/口頭→次回書面）
```

## 共通ルール
- 報告は作業当日中（遅れる場合は「遅れます」だけでも先に送る）
- 数字（件数・枚数）を必ず入れる
- 判断に迷ったものは捨てずに「保留」＋報告に含める
